# Premium Mobile Design Skill

## Core Principles
1. **Depth over Flatness**: Use glassmorphism, depth layering, and subtle 3D effects.
2. **Neon Accents**: Contrast deep navy/black backgrounds with vibrant neon highlights.
3. **Immersive Typography**: Use high-quality fonts like Manrope and Inter with proper hierarchy.
4. **Cinematic Motion**: Implement smooth transitions and micro-animations.

## Flutter Implementation Guidelines
- **Containers**: Use `BackdropFilter` for glass effects.
- **Shadows**: Use highly diffused, ambient shadows instead of sharp ones.
- **Gradients**: Prefer linear gradients with neon endpoints.
- **Rounding**: Use large radii (`BorderRadius.circular(32)` or `StadiumBorder`).
